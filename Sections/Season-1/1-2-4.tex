\SSpsetQ{
    What is the sum of all integers \( x \) such that \( \frac {6969}x \) is an integer?
}\bigskip 

\begin{solution}\hfil\medskip

    Note that if $\frac{6969}x$ is an integer, so is $\frac{6969}{-x}$. So the sum is $\boxed{0}$.
\end{solution}\bigskip
